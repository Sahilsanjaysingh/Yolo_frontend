Safety Object Detection — Documentation

Welcome — this documentation includes endpoints for the demo backend and integration examples.

API

POST /api/upload

multipart/form-data with file field. Returns image document with url.

GET /api/images

List image metadata.

POST /api/chat

Proxy to Gemini. JSON body: { "message": "..." }. Set GEMINI_API_KEY in backend .env.

Frontend integration

// Example upload (fetch)
const form = new FormData();
form.append('file', file);
await fetch('http://localhost:4000/api/upload', { method:'POST', body: form });
